# Machine Learning Projects

Coleção de trabalhos práticos desenvolvidos na disciplina de **Aprendizagem (Machine Learning)** do Mestrado em Engenharia Aeroespacial, com minor em Inteligência Artificial, no Instituto Superior Técnico.

Cada pasta contém um notebook Jupyter completo, com implementação, análise crítica dos resultados e interpretação estatística — não apenas código, mas o raciocínio por trás de cada decisão de modelação.
